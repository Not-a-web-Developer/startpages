### Main Features
- Quick access to your favorite websites
- Supports web-ext (for building the extension)
- Built-in search bar:
    - __When focused on the search bar__:
        - Press `Escape` to move focus away from it
        - Press `Enter` to run the search
    - __When focus is away from the search bar__:
        - Press `s` to focus on it
        - Press `j` to switch to the next search engine (when unfocused from the searchbar)
        - Press `k` to switch to the previous search engine
        - Press `?` to display the current search engine

- The search bar features 5 ways to search the web (ordered):
    - Using Google
    - Using DuckDuckGo
    - Using Github
    - Using StackOverflow
    - Using Reddit
